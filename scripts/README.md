# Utility Scripts

This directory contains utility scripts for development and maintenance tasks.

## Planned Scripts

- **setup_dev.sh**: Set up development environment
- **run_tests.sh**: Run test suite with various options
- **format_code.sh**: Format all code with Black and Ruff
- **check_coverage.sh**: Check test coverage and generate reports
- **setup_gmail.py**: Interactive Gmail API setup wizard
- **test_ollama.py**: Test Ollama connection and models

## Usage

Scripts in this directory are meant to be run from the project root:

```bash
./scripts/setup_dev.sh
```
